- Vos styles doivent être présents dans un dossier “styles”.
- Vos fichiers JavaScript doivent être rangés dans un dossier “js”.
- Vos images, polices... Doivent être placées dans un dossier “assets”.
- Les données doivent être gérées à l’aide d’un fichier JSON.

- La SESSION se fera du coup avec le LOCAL STORAGE vu qu'il n'y a pas de PHP
A chaque ouverture de page on doit donc fetch vers le localstorage pour verifier si une personne est connectée
